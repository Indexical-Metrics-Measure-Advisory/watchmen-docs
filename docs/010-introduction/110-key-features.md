### Ingestion
- Modules — manage system modules with card view (ID, priority, version badge, last modified, quick edit). See [Collector Overview](../080-collector/001-collector-index).
- Models — define ingestion models per module. See [Model Configuration](../080-collector/300-configuration-for-change-data-capture/100-model%20configuration).
- Tables — configure CDC and table-level extraction (keys, audit column, triggers). See [Table Configuration](../080-collector/300-configuration-for-change-data-capture/200-table%20configuration).
- Configuration — global ingestion parameters and runtime behaviors. See [Installation Config · Collector](../060-installation/010-config#collector).
- Monitor — monitor ingestion runtime status and visualize execution flow.
- Actions — top-right controls: `Execution Flow` to visualize DAG, `Create Module` to add modules.

### Admin
- Data Zone — manage logical data zones for isolation and lifecycle. See [Data Zone](../020-web-client/010-admin/010-data-zone).
- Data Source — register and manage data sources. See [Data Source](../020-web-client/010-admin/020-data-source).
- Topic — domain data modeling and factor operations. See [Topic](../020-web-client/010-admin/080-topic).
- Enumeration — shared value sets and relationships. See [Enumeration](../020-web-client/010-admin/070-enumeration).
- Pipeline — canvas-based orchestration with nodes & edges; right panel shows `Incoming Pipelines`, `Outgoing Pipelines`, `External Writers`, and `Factors` count. See [Pipeline](../020-web-client/010-admin/090-pipeline).
- External Writer — output to external systems. See [External Writer](../020-web-client/010-admin/030-external-writer).
- Plugin — extend platform capabilities. See [Plugin](../020-web-client/010-admin/031-plugin).
- Simulator — simulate pipelines and topics with test data. See [Simulator](../020-web-client/010-admin/100-simulator).
- Topic Profile — inspect topic distributions and quality. See [Topic Profile](../020-web-client/010-admin/110-topic-profile).
- Monitor Logs — observe runtime execution and errors. See [Monitor Logs](../020-web-client/010-admin/120-monitor-logs).
- Toolbox — snapshot schedulers, pipeline triggers. See [Toolbox](../020-web-client/010-admin/130-toolbox).
- User — manage users and credentials. See [User](../020-web-client/010-admin/040-user).
- User Group — organize users with permissions. See [User Group](../020-web-client/010-admin/050-user-group).
- Workbench — admin entry and navigation. See [Workbench](../020-web-client/010-admin/001-workbench-index).

### Console
- Home — landing and shortcuts. See [Home](../020-web-client/020-console/010-home).
- Connected Space — organize subjects by business space. See [Connected Space](../020-web-client/020-console/020-connected-space).
- Subject — build datasets via joins, columns, and filters. See [Subject](../020-web-client/020-console/030-subject).
- Report — charts, dataset preview, filters, funnels, variables. See [Report](../020-web-client/020-console/040-report).
- Dashboard — compose multiple reports for monitoring. See [Dashboard](../020-web-client/020-console/050-dashboard).
- Workbench — console entry and navigation. See [Workbench](../020-web-client/020-console/001-workbench-index).

### Data Quality Center (DQC)
- Workbench — DQC entry and navigation. See [Workbench](../020-web-client/030-dqc/001-workbench-index).
- Run Statistics — monitor runtime metrics and performance. See [Run Statistics](../020-web-client/030-dqc/010-run-statistics).
- Monitor Rules — configure quality checks and alerts. See [Monitor Rules](../020-web-client/030-dqc/020-monitor-rules).
- Consanguinity — explore relationships and lineage. See [Consanguinity](../020-web-client/030-dqc/030-consanguinity).
- Catalog — quality assets overview. See [Catalog](../020-web-client/030-dqc/040-catalog).

### Standard Tuple Page
